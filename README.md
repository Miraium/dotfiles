# dotfiles

## 概要

このリポジトリは、私の個人的な設定ファイルを管理するためのものです。  
TmuxとFishを使用したシェル環境を提供します。Nordテーマを採用し、視覚的に快適な作業空間を実現しています。  
さらに、Fishシェル上ではeza、z、fzfといったCLIツールを活用し、作業の生産性を向上させています。  
Python環境の構築にはasdfとpoetryを利用しており、効率的なパッケージ管理が可能です。
